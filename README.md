# IPL Data Analytics Project

## Project Overview

This project analyzes IPL match and ball-by-ball datasets to uncover meaningful insights related to:

- Team performance
- Toss impact
- Player finishing ability
- Stadium behavior
- Match-winning trends

The project was built using Python, Pandas, Matplotlib, and Seaborn.

---

# Objectives

1. Analyze dominant IPL teams
2. Study toss impact on match outcomes
3. Identify the best IPL finishers
4. Analyze batting-friendly stadiums
5. Generate visual insights using data analytics

---

# Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# Key Insights

- Mumbai Indians and Chennai Super Kings are among the most dominant IPL teams historically.
- Winning the toss provides an advantage but does not guarantee victory.
- Death-over specialists maintain exceptional strike rates under pressure.
- Certain stadiums consistently produce high-scoring matches.
